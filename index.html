QUESTION : How to Apply CSS to Html ?

ANSWER : To include an external CSS file in an HTML file, we can use the <link> Tag within the <head> section. For example: 

                    <head>
                    <link rel="stylesheet" href="style.css">
                    </head>

QUESTION : How to Use JavaScript in a Webpage ?

ANSWER : To link to an external JavaScript file, we can use the <script> tag. For example: 

                <script src="script.js"></script>

To write JavaScript code in an HTML file, we can use the <script> tag within the <head> or <body> section. For example: 

                <script>
                //your JavaScript code
                </script>
